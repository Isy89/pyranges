7 Piping PyRanges

The PyRanges API supports chaining operations out of the box. It is often terser than the equivalent in pandas or dplyr.

import pyranges as pr
import pandas as pd
exons = pr.data.exons()
cpg = pr.data.cpg()
cpg.join(exons.unstrand())(lambda df: df.CpG > 25)[["CpG"]](lambda df: df.head(3))["chrX"].slack(500)

To debug pipelines, you can use the experimental pr.sandbox.Debug(). Since the debugging context-manager reads source files, it needs to be used in a file, not a REPL. See piedpiper for more info.

import pyranges as pr
from pyranges.sandbox import Debug as D

exons = pr.data.exons()
cpg = pr.data.cpg()

with D():
    cpg.join(exons.unstrand())(lambda df: df.CpG > 25)[["CpG"]](lambda df: df.head(3))["chrX"].slack(500)
