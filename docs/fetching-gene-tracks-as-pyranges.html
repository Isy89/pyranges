23 Fetching gene tracks as PyRanges

PyRanges can fetch data from UCSC, GENCODE and Ensembl through the pyranges.db module.

Each database contains at least two methods, namely genes and genomes. UCSC and Ensembl also has the method chromosome_sizes.

The genes-methods all take a boolean flag head If true, it only fetches a few hundred rows of data.

import pyranges as pr
genomes = pr.db.ucsc.genomes()
print(genomes.head())
##     Genome
## 0  acaChl1
## 1  ailMel1
## 2  allMis1
## 3  allSin1
## 4  amaVit1
chr_sizes = pr.db.ucsc.chromosome_sizes("hg38")
print(chr_sizes.head())
## chr1    248956422
## chr2    242193529
## chr3    198295559
## chr4    190214555
## chr5    181538259
## dtype: int64
genes = pr.db.ucsc.genes("hg38", head=True)
print(genes)
